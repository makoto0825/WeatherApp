About this App

  • Check the weather

    The current weather conditions are presented.

  • More detail

    Detailed information on the weather, including temperature, humidity, wind speed, and more, is provided.

  • Clothes recommendation

    User can get appropriate clothing suggestions based on the current weather is provided.

  • Setting

    User can set the region. Additionally, user can also switch between casual and formal clothes suggestions.
